該套件的配置有分成兩種模式，使用者可以自行決定要用哪一種方式來進行配置。

  • 簡單配置 （支持三種模式)
  • 高級配置 （客製化的參數設定）

官方說明文件: hexo-renderer-markdown-it

簡單配置

簡單設置有提供三種模式，讓使用者可以輕鬆設置

  • zero：大多解析功能不支援。
  • default（GFM）：支援github的規範解析。
  • commonmark： 提供更多的預設功能，同時將強制插件使用最新的CommonMark規範．
